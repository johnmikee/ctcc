# ctcc
see your tcc db

This is a simple port of [tcc](https://github.com/Mac-Nerd/tcctool/blob/main/tcctool.sh) to go with a few extra features.

## Usage
```sh
./ctcc -h             
Usage of ./ctcc:
  -csv
        Output to CSV
  -file string
        File name to output to (default "ctcc.csv")
  -json
        Output to JSON
  -sal
        Output to Sal
  -version
        Print version and exit
```
